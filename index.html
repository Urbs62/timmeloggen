Instructions

Track time and generate invoices in minutes.

1) Go to Invoice Settings
Add company details, hourly rate, VAT, currency and set invoice number

2) Add Accounts / Projects
Create one or multiple accounts per customer

3) Log time
Add entries and assign to accounts (start/end time optional)

4) Open Summary
Select month and set monthly target (optional)

5) Generate invoice or timesheet
Invoice number is taken from Settings
